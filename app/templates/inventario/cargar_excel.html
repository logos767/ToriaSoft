Cargar Inventario desde Excel

Instrucciones

Para cargar tus productos, tu archivo de Excel debe tener el siguiente formato de columnas, sin encabezados:

  • Columna 1: Código de Barras (Obligatorio, para identificar el producto)
  • Columna 2: Código del Producto (Opcional)
  • Columna 3: Nombre del Producto (Obligatorio)
  • Columna 4: Costo (USD) (Obligatorio)
  • Columna 5: Precio de Venta (USD) (Obligatorio)
  • Columna 6: Existencia a Cargar (La cantidad que se sumará al almacén seleccionado)
  • Columna 7: URL de la Imagen (Opcional)
  • Columna 8: Marca (Opcional)
  • Columna 9: Color (Opcional)
  • Columna 10: Talla (Opcional)

* Se recomienda usar el formato .xlsx.
